🏢 INSTALLATIONS BUSINESS APP - WEB APPLICATION
==============================================

📦 Was ist das?
Ein komplettes webbasiertes Angebotssystem für Installationsbetriebe

🚀 Zugang:
Die App läuft als Web-Anwendung über Railway und ist direkt im Browser verfügbar.

💻 Systemvoraussetzungen:
- Moderner Browser (Chrome, Firefox, Edge, Safari)
- Internetverbindung

🆘 Bei Problemen:
Support kontaktieren

✨ Hauptfunktionen:
- Kundenverwaltung
- Angebotserstellung mit Arbeitspositionen
- PDF-Export
- Auftragsverwaltung
- Lieferantenbestellungen
- Datensicherung

Version 1.0 - Juli 2025
